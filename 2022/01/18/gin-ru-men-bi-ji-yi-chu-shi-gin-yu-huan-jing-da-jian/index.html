一、Gin 是什么

Gin 是一个用 Go (Golang) 编写的 web 框架。 它是一个类似于 martini 但拥有更好性能的 API 框架, 由于 httprouter，速度提高了近 40 倍。

同时，它目前在 GitHub 上已经有了 50k+ 的 Star，可谓是非常热门
